## 安装
```shell
# arch 
sudo pacman -S awesome
# ubuntu
sudo apt install awesome
# arch/manjaro 安装
git clone https://github.com/denstiny/awesome-config.git 
bash awesome-config/src/install.sh
# 其他linux 请根据脚本手动安装
git clone https://github.com/denstiny/awesome-config.git 
```
